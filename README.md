# Image to Material - Project README

写真の遠近歪みを補正してシームレスなテクスチャ素材を生成するWebアプリケーション

## 概要

Picture To Texture は、斜めから撮影した写真の遠近歪みを補正し、正面から見た長方形のテクスチャ画像を生成するツールです。ホモグラフィ変換を使用して高品質な画像変換を実現しています。

## 機能

- 📤 **画像アップロード**: ドラッグ＆ドロップまたはファイル選択で画像を読み込み
- 🎯 **4点選択**: 補正したい領域の4つの角を選択
- 🔄 **リアルタイム変換**: ホモグラフィ変換によるリアルタイムプレビュー
- 📏 **サイズ調整**: 出力画像のサイズを自由に設定
- 🔒 **縦横比ロック**: アスペクト比を固定したサイズ調整
- 💾 **フォーマット選択**: PNG (高品質) または JPEG (軽量) で保存
- ⚙️ **画質設定**: JPEG の圧縮率を調整可能
